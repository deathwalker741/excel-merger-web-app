What This Tool Does

This web application processes Excel files containing school data and intelligently merges duplicate records based on the School Number. It's designed specifically for educational data management where multiple entries for the same school need to be consolidated.

Fields That Get Summed

The following fields are automatically aggregated (summed) when merging duplicate school records:

  • Total Order Value (Exclusive GST)
  • Total Order Value (Inclusive GST)
  • ASSET Revenue
  • ASSETStudents
  • CARES Revenue
  • CARESStudents
  • Mindspark Revenue
  • MindsparkStudents

How It Works

Step 1: Upload your Excel (.xlsx) file containing school data

Step 2: The system groups all rows by "School No"

Step 3: For each group:

  • Financial fields are summed together
  • Text fields are merged (identical values kept as-is, different values combined with commas)
  • School numbers are preserved

Step 4: Download the processed file with merged data

File Requirements

Your Excel file should:

  • Be in .xlsx format
  • Have a column named "School No" for grouping
  • Include the financial columns you want to aggregate
  • Have proper headers in the first row
